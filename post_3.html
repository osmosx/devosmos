Webpack - сборщик модулей JavaScript с открытым исходным кодом

Webpack это многофункциональный сборщик модулей. Основная его задача - компоновать JavaScript код в отдельные файлы для создания модульной архитектуры. Кроме компоновки JavaScript кода, Webpack также может преобразовывать HTML и CSS-код, работать с изображениями и многое другое. Для него написано большое количество плагинов, которые можно подключать к себе в проект. Инструкция по основам настройки проекта на JavaScript и Sass есть на сайте Tproger. Ну и конечно руководство для начинающих есть на Habr.
